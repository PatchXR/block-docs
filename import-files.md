Supported file types: .wav, .ogg, .mp4, .avi, .mov, .jpg, .jpeg .png, .gif, .glb and .patch (100mb max) 
If you need to import larger files contact the team on Discord.

Uploading files is done via the portal.

1. Navigate to portal.patchxr.io
2. Login 
3. In the nav bar click Upload 
4. Drag and drop files or click to browse and select
