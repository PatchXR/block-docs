# Spawning blocks 
1. Start a new world, then open the main menu and navigate to the "Create" tab.
2. Select "Blocks" from the menu on the right.
3. Click on a block to spawn it via the Spawn button

# Connecting blocks
You can connect blocks together to create logic, interactions and more!
Make sure to get familiar with [the connection types used](/patching/connections).

# Block Documentation 
Each block will have description and also description of its subparts displayed as tooltips in the app.
Hover over a block with the tip of the controller to see the tooltips.

The same documentation is available here in the wiki: [Blocks Documentation ](/blocks)
