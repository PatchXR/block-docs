## Description

A resizable trigger box which can detect if a player is inside. Can also be used to attached player to a vehicle's cockpit/seat.

## Inputs, Outputs and Parts

**Player inside** *(jolt output)*: Emits the number of players inside the box.

**Resize handle** *(interactive)*: Grab and move to resize the trigger area.
