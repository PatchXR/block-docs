## Description

Outputs all players. (also nearest player / local player / ghosts)

## Inputs, Outputs and Parts

**On Player Joined**: Outputs total player count. Triggered when a new player join the session.

**On Player Left**: Outputs total player count. Triggered when a player is leaving the session.

**Local Player**: Youself. Warning: in Multiplayer session, every player will see himself on this output!

**All Players**: Outputs all players, including yourself.

**All Ghosts**: Outputs all "ghosts" in the room.  ... *boo!*

**Nearest Player**: Nearest Player to this block.
