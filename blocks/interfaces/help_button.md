## Description

Push the button to change the state, emitting 0 or 1 when switching states.

Useful for activating help texts, or ghost recording tutorials.

## Inputs, Outputs and Parts

**State Out**: Emits 0 or 1 depending on the button state.

**Set State**: Sets the state of the button with 0 (off) and 1 (on).

**Highlight**: Controls the brightness of the button graphics from -1 (black) to 4 (white).
