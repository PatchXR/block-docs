## Description

Used to attach two blocks together and let you set the distance between the two (to make suspension, piston, spring etc)
- Place this joint inside a physical group.
- Connect the selector to a different group (or block).
Both group will be subject to the constraint.
Activate the different features from the Inspector.

## Inputs, Outputs and Parts

**Connected Block**: Connected block will become automatically physical if no "Make Physical" block is used. Can connect to only one block or group.

**Piston Size**: Only affects visuals.
