## Description

Change position of the "target" block. You can give an optional "origin" block, it will then position the target block relative to this one.

## Inputs, Outputs and Parts

**Origin (optional)**: Target position will be set relative to this block.

**Target**: Block to Set Position.

**X**: Move Left/Right.

**Y**: Move Up/Down.

**Z**: Move Back/Forward.
