## Description

Get X,Y,Z position of the connected block ("target" input). Can also get position relative to another block ("Origin" optional input).

## Inputs, Outputs and Parts

**X**: Left/right

**Y**: Down/up

**Z**: Back/front

**Origin (Optional)**: Output position will be relative to this block if connected.

**Target**: Block to Get Position.
