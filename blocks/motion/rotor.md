# Rotor

**Category**: 🚀 Motion

![rotor thumbnail](https://portal.patchxr.io/block-thumbnails/rotor.png)

## Description

Attach other blocks to it using click and drag. Changing its input value rotates them.

## Inputs, Outputs and Parts

### Inputs

| Name | Type | Description |
|------|------|-------------|
| Rotate | Stream Input | Rotates attached blocks. Input value is multiplied by 10 to get degrees (e.g., 36 = 360 deg, 18 = 180 deg). Negative values rotate in the opposite direction. |
| Attached Blocks | Tag Input | Click and drag to attach block and rotate them. |
