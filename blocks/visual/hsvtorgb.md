## Description

Converts HSV color values to RGB format. Input hue, saturation, value and get red, green, blue outputs.

Color space conversion utility that transforms HSV (Hue, Saturation, Value) color representation into RGB (Red, Green, Blue) format. Useful for color calculations and interfacing between different color systems. All values operate in 0-1 range.

## Inputs, Outputs and Parts

**Red Output**: Red component of RGB color (0-1 range)

**Green Output**: Green component of RGB color (0-1 range)

**Blue Output**: Blue component of RGB color (0-1 range)

**Hue Input**: Hue value input (0-1 range, represents color wheel position)

**Saturation Input**: Saturation value input (0-1 range, color intensity)

**Value Input**: Value/brightness input (0-1 range)
