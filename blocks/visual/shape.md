# Shape

**Category**: 🎨 Visual

![shape thumbnail](https://portal.patchxr.io/block-thumbnails/shape.png)

## Description

Configurable primitive shape: sphere, cube, pyramid, diamond, cylinder, torus, cone, disk, screen, and more. Pick the mesh via the inspector.

A single block that can become any of ~30 primitive meshes (spheres, cubes, cylinders, polygons, cones, torus, rings, disks, screens, etc.). Use the inspector preset selector to step through the catalogue. The block's colour tints the mesh, and a collider matching the selected shape is set up automatically.

## Inspector Controls

| Name | Type | Description |
|------|------|-------------|
| Sphere | select_button | Cycles through the available primitive meshes (sphere, cube, cylinder, torus, etc.). The selected mesh also drives the collider shape. |
