## Description

Changes the size of target blocks along X, Y, and Z axes. Drag selector to blocks and adjust scale knobs.

Scale manipulation tool with separate X, Y, and Z axis controls. Use the interactive selector to target blocks, then adjust size independently on each axis using dedicated knobs. Values typically range from 0.1 to 10, with 1 being original size.

## Inputs, Outputs and Parts

**Target Selector**: Drag to blocks to select scale targets

**Scale X**: Controls horizontal (width) scaling

**Scale Y**: Controls vertical (height) scaling

**Scale Z**: Controls depth scaling
