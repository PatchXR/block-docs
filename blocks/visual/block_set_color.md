## Description

Changes the color of target blocks using HSV controls. Drag selector to blocks and adjust hue, saturation, brightness knobs.

Color manipulation tool with HSV (Hue, Saturation, Brightness) controls. Use the interactive selector to target blocks, then adjust color properties with dedicated knobs. Input values should be in 0-1 range. Changes apply in real-time as you modify the parameters.

## Inputs, Outputs and Parts

**Target Selector**: Drag to blocks to select color targets

**Hue**: Controls color hue (0-1 range)

**Saturation**: Controls color saturation intensity (0-1 range)

**Brightness**: Controls color brightness/value (0-1 range)
