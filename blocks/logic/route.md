## Description

Let you route a signal with different behaviors: based on given index, loop (one after another), random, all. (sequence, dispatch, order, select, split)

## Inputs, Outputs and Parts

**Input**: Send your value here.

**Reset**: Reset the index to zero. (loop mode)

**Index**: Will select which output will be triggered.
