# Jolt Pass

**Category**: 🔗 Connectors

![pass thumbnail](https://portal.patchxr.io/block-thumbnails/pass.png)

## Description

Passes a jolt event from input to output without modification. Useful as a junction point to route or organize wires in complex patches.

## Inputs, Outputs and Parts

### Inputs

| Name | Type | Description |
|------|------|-------------|
| Pass In | Jolt Input | Jolt input. To connect another block here, click & pull on any Jolt output to create a wire and drag it here. |

### Outputs

| Name | Type | Description |
|------|------|-------------|
| Pass Out | Jolt Output | Jolt output that re-emits whatever is received on Pass In. Click & pull to generate a new Jolt wire. |
