# Jolt Out

**Category**: 🔗 Connectors

![pass_out thumbnail](https://portal.patchxr.io/block-thumbnails/pass_out.png)

## Description

Passes a jolt signal through with no processing. Used inside a group or device to visually mark where external jolt outputs go out.

| Name | Type | Description |

| Input | Jolt Input | To connect another block here, click & pull on any Jolt output to create a wire and drag it here. |

### Outputs

| Name | Type | Description |
