# Load Counter

**Category**: ⚙️ System

![launch_counter thumbnail](https://portal.patchxr.io/block-thumbnails/launch_counter.png)

## Description

Count how many times you loaded this world. (specific to individual users). First visit is zero.

## Inputs, Outputs and Parts

**Get count**: Will trigger the output with the load counter value.

**Out**: Trigger when world is loaded.
