# Block Spawn

**Category**: ⚙️ System

![block_spawn thumbnail](https://portal.patchxr.io/block-thumbnails/block_spawn.png)

## Description

You can give it a block or device as reference, and spawn copies of this same block.

You can give a life time to spawned blocks.
This block also output all spawned blocks.
