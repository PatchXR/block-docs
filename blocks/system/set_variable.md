# Set Variable

**Category**: ⚙️ System

![set_variable thumbnail](https://portal.patchxr.io/block-thumbnails/set_variable.png)

## Description

Given a "VariableName" and a value, it can assign a variable to a block. Helps you then collect and sort all blocks with specific value using "All With Variable" for gameplay! examples: Score, Health, IsRocket.

## Inputs, Outputs and Parts

**Value**: Is setting value to all connected blocks.

**Input**: Blocks we want to add a variable to.

**Restrict (Optional)**: Restricts to a group: Variable will become local to this group, "get variable" & "all with variable" will need to have the same Restrict Input to be able to read the value.

**Manual Trigger**: If connected: Trigger the block
If not connected: block trigger automatically when Input or Value changes.
