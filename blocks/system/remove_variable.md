# remove_variable

**Category**: ⚙️ System

![remove_variable thumbnail](https://portal.patchxr.io/block-thumbnails/remove_variable.png)

## Description

Remove a variable attached to the given block.

## Inputs, Outputs and Parts

**Input**: Blocks we want to remove a variable from.

**Manual Trigger**: If connected: Trigger the block
If not connected: block will trigger automatically.

**Restrict (Optional)**: Restricts to a group: necessary if "set variable" uses it.
