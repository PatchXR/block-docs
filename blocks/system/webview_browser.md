# Web Browser

**Category**: ⚙️ System

![webview_browser thumbnail](https://portal.patchxr.io/block-thumbnails/webview_browser.png)

## Description

Embeds a web browser into the VR environment with interactive navigation controls.

Displays web pages in a 3D canvas that can be resized and controlled. Includes standard browser functions like back/forward navigation, refresh, zoom, and URL input. Audio from web content can be enabled but does not route through the synthesis audio engine. Keyboard input is supported for text fields.

## Inputs, Outputs and Parts

**Back**: Navigate to the previous page in browser history

**Forward**: Navigate to the next page in browser history

**refresh**: Reload the current page

**Enable Audio**: Toggle audio playback from web content (audio bypasses synthesis engine)

**Zoom**: Adjust the page zoom level

**download**: Download files from the current page

**open_keyboard**: Open virtual keyboard for text input

**Resize**: Drag handle to resize the browser window

**URL**: Enter web address to navigate to

**Canvas**: The web page display area
