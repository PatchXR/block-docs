# Block Remove

**Category**: ⚙️ System

![block_remove thumbnail](https://portal.patchxr.io/block-thumbnails/block_remove.png)

## Description

Remove the given block. A delay can be set.

## Inputs, Outputs and Parts

**Input**: Blocks we would like to remove.

**Remove**: Trigger the removal, with a delay if any delay is set.

**Delay**: Set a delay from the moment "Remove" input is triggered and the block actually disappear.
