# Join & Apply Text

**Category**: ⚙️ System

![join_apply_text thumbnail](https://portal.patchxr.io/block-thumbnails/join_apply_text.png)

## Description

Given multiple blocks with text, it combine the different text into one and apply to its target block.

## Inputs, Outputs and Parts

**Source Text**: Can have multiple text as input, so it joins all text together.

**Manual Trigger**: You can connect a jolt here so it doesn't automatically trigger itself when source blocks change.

**Target block**: Target blocks, whose text will change.

**Clear**: Clear the text.
