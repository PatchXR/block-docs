# Preset Manager

**Category**: ⚙️ System

![preset_manager thumbnail](https://portal.patchxr.io/block-thumbnails/preset_manager.png)

## Description

Will save the state of every connected blocks. It can be triggered to switch between different presets.

## Inputs, Outputs and Parts

**Restore Values**: Should it restore the state and values of elements when a preset is triggered ?

**Restore Position**: Should it restore the position of elements when a preset is triggered ?

**Restore Connections**: Should it restore the connections of elements when a preset is triggered ?

**Restore Visibility**: Should it restore the visibility of elements when a preset is triggered ?

**Transition time**: How long (in seconds) to reach the preset position when preset is triggered.

**Transition time**: How long (in seconds) to reach the preset position when preset is triggered.

**Add/Remove preset**: Click, or drag me down to make new presets.

**Add blocks**: Drag me to other blocks to add them to the preset.

**Preset Select**: Combined with the other jolt input, you can select a preset before sending an operation instruction (like activating the selected preset).

**Preset Operation**: Trigger operation on the selected preset (Preset Select input)
  0: Apply the preset to the subjects
  1: Write the preset (saved subject's states in the preset)
  2: Deleted the preset.
