# Block Compare

**Category**: ⚙️ System

![block_compare thumbnail](https://portal.patchxr.io/block-thumbnails/block_compare.png)

## Description

Compare two list of blocks and output a third one:
==: A & B the same ?
A-B: all from A which are not in B
Shared: only blocks present in both lists
Grabber: output player grabbing the block
Store: store block from input until "Clear" is trigger
