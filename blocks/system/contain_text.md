# Contain Text

**Category**: ⚙️ System

![contain_text thumbnail](https://portal.patchxr.io/block-thumbnails/contain_text.png)

## Description

Checks if the given text blocks contain a specific word or pattern in their content and outputs the occurrence count of the search.

## Inputs, Outputs and Parts

**Search pattern**: The pattern to search in the given text.

**Source Text**: The source text to search in.

**Manual Trigger**: You can connect a jolt here so it doesn't automatically trigger itself when source blocks change.

**Occurence Result**: Output 0 to N number depending on the occurences found.
