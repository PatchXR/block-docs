# Device Landmark

**Category**: ⚙️ System

![landmark thumbnail](https://portal.patchxr.io/block-thumbnails/landmark.png)

## Description

When placed inside a device, it tells where is the center of this device, and where is its front.
It helps to get the device well placed when spawned from library, also acts as pivot point for kinetic blocks.
