# set_dsp_enabled

**Category**: ⚙️ System

![set_dsp_enabled thumbnail](https://portal.patchxr.io/block-thumbnails/set_dsp_enabled.png)

## Description

Enable or disable audio processing for selected blocks.

## Inputs, Outputs and Parts

**Selector** *(selector)*: Selects which blocks/groups to enable or disable.

**DSP Enable** *(jolt input)*: Send a 1 to enable DSP processing, 0 to disable.
