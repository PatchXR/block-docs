# Get Name Text

**Category**: ⚙️ System

![get_name thumbnail](https://portal.patchxr.io/block-thumbnails/get_name.png)

## Description

Get the name of the given blocks. (can also give player's or device's name) If multiple blocks are given, it will separate them with a comma and whitespace.

## Inputs, Outputs and Parts

**Source Block**: Blocks from which we retrieve the name.
