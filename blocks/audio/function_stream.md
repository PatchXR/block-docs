# Function Stream

**Category**: 🔊 Audio

![function_stream thumbnail](https://portal.patchxr.io/block-thumbnails/function_stream.png)

## Description

Applies a math function continuously to a stream signal. Functions: sin, cos, tanh, floor, round, abs, sign, to dB, from dB, MIDI-to-frequency (m2f), last, vel.

## Inputs, Outputs and Parts

### Inputs

| Name | Type | Description |
|------|------|-------------|
| Input | Stream Input | Stream signal to apply the function to. |

### Others

| Name | Type | Description |
|------|------|-------------|
| Operation | Selector | Click & drag the dial to choose between different functions: 0: Sin, 1: Cos, 2: Floor, 3: Round, 4: Abs, 5: Sign, 6: to dB, 7: from dB, 8: Tanh, 9: M2F (MIDI to Hz), 10: last, 11: vel. |
