# Bow

**Category**: 🎮 Controllers

![bow thumbnail](https://portal.patchxr.io/block-thumbnails/bow.png)

## Description

A bow-shaped grabbable controller, intended to be rubbed against the Violin String block (e_string) to drive its output.

Grab and move the bow across a Violin String to play it. The string block reads the bow's contact point, pressure, and velocity to generate signals you can wire into audio.
