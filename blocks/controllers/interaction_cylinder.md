## Description

Like the interaction_box, but cylinder shaped.

## Inputs, Outputs and Parts

**Resize handle** *(interactive)*: Use trigger to resize.

**Left controller height** *(jolt output)*: Sends out the left controller height

**Left controller radius** *(jolt output)*: The distance of the left controller to the center of the cylinder.

**Left controller angle** *(jolt output)*: The angular position of the controller. Range 0 to 1 clockwise.

**Left controller present** *(jolt output)*: Sends a 1 when a left controller enters and 0 when it exits.

**Left controller trigger** *(jolt output)*: Sends out the analog value of the left controller's trigger button.

**Left controller roll** *(jolt output)*: Sends out the roll angle of the left controller. Range -0.5 to 0.5.

**Left controller haptics** *(stream input)*: Controls the vibration of the left controller.

**Left controller haptics** *(stream input)*: Controls the vibration of the left controller.

**Right controller present** *(jolt output)*: Sends a 1 when a left controller enters and 0 when it exits

**Right controller trigger** *(jolt output)*: Sends out the analog value of the left controller's trigger button.

**Right controller roll** *(jolt output)*: Sends out the roll angle of the left controller. Range -0.5 to 0.5.

**Right controller height** *(jolt output)*: Sends out the right controller height

**Right controller radius** *(jolt output)*: The distance of the right controller to the center of the cylinder.

**Right controller angle** *(jolt output)*: The angular position of the controller.

**Right controller haptics** *(stream input)*: Controls the vibration of the left controller.

**Right controller haptics** *(stream input)*: Controls the vibration of the left controller.

**Num. height segments** *(knob)*: Controls the number of segments along the height of the cylinder.

**Num. radial segments** *(knob)*: Controls the number of segments going from the middle of the cylinder outwards.

**Num. angular segments** *(knob)*: Controls the number of segments going around the cylinder (like a clock).
