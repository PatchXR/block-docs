## Core Concepts

-   [**Blocks**](/patching/intro-to-blocks) - The basic building blocks of PatchWorld
-   [**Connections**](/patching/connections) - How blocks communicate
    -   [Streams](/patching/connections#Stream) - Continuous values at sample rate
    -   [Jolts](/patching/connections#Jolt) - Single value bursts
    -   [Tags](/patching/connections#Tag) - Data connections between blocks
-   [**Groups and Devices**](/patching/groups-and-devices) - Organizing blocks together
-   [**Assets**](/patching/assets) - Images, Video, 3d models, Audio
-   [**Ghost Recordings**](/patching/ghosts) - Recording your avatar voice/movements/interactions
-   [**Worlds**](/patching/worlds) - PatchWorld projects and collections
