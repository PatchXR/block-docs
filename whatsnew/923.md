# 🚀 PatchWorld Update: Versions 77.2.1 & 77.2.2

Hey everyone! A massive wave of improvements, new features, and fixes has just landed in the universe. Here is everything that’s changing for you in this version:

### ✨ New Features & Highlights

* **🖼️ New Block: Image Downloader!** You can now load and display images directly onto an image block (via URL, user profile, or asset).
* **🌌 Snapshot Evolution:** The Snapshot block can now capture the **depth map** of the environment. Get ready for some advanced visual effects!

### 🛠️ Edit Mode & UI (QoL)

* **🔍 Direct Library Access:** Added a **"See In Library"** (+ More Info) button directly in the inspector for every single block.
* **📐 Easier Grid Tweaks:** You can finally access Grid settings via the radial menu, even when you are inside a Group.
* **🔒 Clearer Edit Mode:** "Allow Editing" is now **"Allow Edit mode"**. It prevents entering edit mode or opening groups, and fixes text input interactivity issues on disabled remixes.

### ⚙️ Physics & Block Behaviors

* **🎯 Customizable Look At:** The *Look At* block now lets you select which specific axis points at the target (`+x`, `-x`, `+y`... default is `+z`).
* **📦 Enhanced Trigger Box:** It can now filter players and output the list of players currently inside. Perfect for making a personalized welcome message in your world!
* **📡 Wireless Variables:** The *set_variable* block gets a **"Remove On Disconnect"** option. It automatically removes the variable when the block is disconnected, allowing it to be used as a wireless "Tag Send / Tag Receive" system with *all_with_variable*.
* **🌐 MP Physics Sync:** *Make Physical* now allows you to adjust the syncing rate in multiplayer.

### 🖥️ Console Commands (For Power Users)

* **🕶️ `SetFoveating True/False`:** A brand new command to toggle Foveated Rendering (Shoutout to @Hanumanatee for the request!).
* **📚 `OpenAssetInLibrary(assetID)`:** Open any specific asset directly in the library using its ID.

### 🐛 Bug Fixes

* **☀️ Graphics:** Shadow support is officially back!
* **🌀 Portals:** Fixed the world portal scaling issue where the title would weirdly move away.
* **🕹️ Controls:** Fixed the Device Handle inconsistency between enabling X/Y joystick axes in the inspector and its actual behavior.
* **🧩 Grid Physics:** Fixed a slight offset issue when moving a joint with attached physics on the grid while in edit mode.
* **🪢 Jolt Interaction (v77.2.2):** Fixed an annoying bug where stream inputs would catch Jolt wires and refuse to let go!
