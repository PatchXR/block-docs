## вңЁ Headline Features & New Blocks

### рҹҢҗ New Block: Web Bridge
Bring the full power of the internet into your VR worlds! The brand-new **Web Bridge** block lets you embed custom web pages and use JavaScript to create dynamic, two-way interactions between your PatchWorld creations and external web apps.

### рҹ“қ Text Processor (All-in-One Text Block)
Say goodbye to messy text logic chains! We've evolved the *Join & apply text* block into the powerhouse **Text Processor**. It consolidates everything into a single super-block supporting `Join`, `Split`, `IndexOf`, `Regex`, `SubString`, and more.

### вҡЎ New Jolt Event: `onspawn`
Building reusable marketplace instruments or devices? The new `onspawn` Jolt event fires whenever a group containing a **Receive Jolt** block is spawned from the library or duplicated. Unlike `onload`, it won't trigger during regular world reloadsвҖ”making it the perfect trigger for clean initialization logic!

## рҹӣҚпёҸ Marketplace Economy & Creator Protections

We are giving creators more peace of mind while making discovering and testing community assets much smoother:

- **рҹҺҒ "Try Before You Buy" Mode**: 
  - Creators can now enable a free trial mode on their Marketplace assets! 
  - By default, players get **3 free tries** per asset (creators can customize this number).
  - Assets spawned in Trial Mode can be fully experienced in-world, but cannot be saved, copied, opened, or inspected.
- **рҹӣЎпёҸ Purchase Protection**: Added verification to prevent unauthorized duplication of marketplace assets you haven't purchased.
- **рҹ’ҫ World Saving Verification**: Asset ownership verification is now properly saved within world patches (previously this only applied to standalone Devices).

## рҹҺЁ Visuals, Meshes & Materials

- **вҳҒпёҸ 3D Mesh Cloud Reloading**: Iterating on custom 3D models? You can now instantly reload 3D meshes from the cloud directly via the Inspector whenever you update them on the web portal. Zero world reloads needed!
- **рҹ“җ Single vs. Double-Sided Meshes**: You can now explicitly force 3D Meshes to render as single- or double-sided in the Inspector. *(Tip: Keep meshes single-sided on Quest to save GPU cycles!)*
- **рҹҸғ Collisions on Animated Meshes**: Added support for physics collisions on animated meshes, alongside a new input on `make_physical_extra` to manually trigger collider geometry updates.
- **рҹҺЁ Set Material Upgrades**: 
  - Added a new `Select Material` Jolt input for easier programmatic triggering.
  - Added a **"Keep Sections Open"** Inspector setting so you can browse multiple material sections simultaneously without them collapsing.
- **рҹ–јпёҸ Crisp Portal Previews**: Doubled the resolution of world portal preview images from 256px to 512px for sharper browsing.
- **рҹ“– Extended Library Tooltips**: Detailed descriptions for Inspector settings are now displayed directly inside the Block Info view in the Library.

## вҡҷпёҸ Gameplay, Logic & Multiplayer Fixes

- **рҹ‘Ҙ Ultra-Responsive Player Detection**: The `players` block now scans for the nearest player every **0.3 seconds** (up from 1.0s) for much tighter proximity interactions.
- **рҹ“Ў Cloud API Quota Boost**: Increased the rate limit for Fetch / Post Cloud blocks to **20 operations every 10 seconds** (shared across all cloud blocks in your world).
- **вҡЎ Receive Jolt Default**: Newly spawned Receive Jolt blocks now include `onload` in their channel list by default.
- **рҹҺҜ Recursive Set Interactive**: The `Set Interactive` block now applies to groups and nested subpatches recursively.
- **рҹӘ„ Tag Connection Cabling Fix**: Pulling Tag connection lines while holding the trigger now properly connects and affects the target block upon releasing the trigger.
- **рҹ”Ё Physics Collision Fix**: Fixed a bug in `make_physical_extra` where the assigned "in charge" player remained stuck after a collision ended.
- **рҹ’» New Console Command (`CountBlocks`)**: Type `CountBlocks` in the console to get an instant tally of all active blocks and groups in your world.
- **рҹ’» New Console Command (`CreateOrJoinMultiplayer`)**: Type `CreateOrJoinMultiplayer` to instantly open or join a multiplayer session in your current world without triggering a scene reload.
