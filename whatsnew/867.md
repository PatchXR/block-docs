# PatchWorld 76.8 Release Notes

**What's new since version 75.2 (853)**

## üéÅ Major New Features

### Variable System
Store and retrieve values across your world with the new variable blocks:

- **Set Variable** ‚Äî Assign values to blocks like wireless connections, but for tags/selectors
- **Get Variable** ‚Äî Read variable values from any block
- **Remove Variable** ‚Äî Clear a variable from a block
- **All With Variable** ‚Äî Find all blocks with a specific variable or value

### Online Variables
Connect your worlds to the cloud:

- **Post Variable** ‚Äî Send text or values to the server
- **Fetch Variable** ‚Äî Download values or text from the server

### New Browser Block
Embed web content directly in your worlds with full interactivity. Features adjustable resolution, URL navigation, volume/zoom knobs, and scaling handles. Found in Visuals/Media category.

### Players & Multiplayer System

- **Players Block** ‚Äî Get all players in a live session to perform individual actions on each
- **Player Body Parts** ‚Äî Players and ghosts now made of 5 targetable parts (origin, head, hands, feet)
- **Hide Player Parts** ‚Äî Use Set Visible to hide specific body parts
- **Target Specific Players** ‚Äî Player Position, Haptic Control, and Controller Output blocks can target individuals

### New Gameplay Blocks

- **Tag Gate** ‚Äî Disconnect/reconnect tag connections via jolt input or toggle
- **RayCast** ‚Äî Detect objects in front with adjustable distance, thickness, and layer filtering
- **Block Remove** ‚Äî Make blocks disappear
- **Block Compare** ‚Äî Compare two lists of blocks (combine, subtract, find shared, check equality)
- **Get SubPart** ‚Äî Select specific parts of players or meshes

### Bubble Player & Recorder
New visual blocks for audio recording/playback ‚Äî now colorizable with configurable player selection.

### Library 4.0
A completely redesigned library experience:

- **Featured** ‚Äî Admin-curated content and collections
- **Published** ‚Äî Browse and pin favorite creators
- **My Assets** ‚Äî Purchases, likes, custom folders, and spawn history
- **News & Wiki** ‚Äî Blog and documentation access
- Instant search with filters and sorting
- Drag-and-drop organization with multi-select and bulk actions
- Custom folder icons and up to 4 levels of nesting
